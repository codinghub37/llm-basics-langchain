# LLM Basics with LangChain

This project demonstrates the fundamentals of working with Large Language Models (LLMs) using LangChain. It covers prompt engineering, system messages, temperature tuning, and interaction with multiple LLM providers, including Gemini and Groq.

## Features

- LLM Invocation
- Prompt Engineering
- System Messages
- Temperature Tuning
- LangChain Integration
- Gemini & Groq Models

## Tech Stack

- Python
- LangChain
- Google Gemini
- Groq
- Jupyter Notebook
